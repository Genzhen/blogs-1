## 团队建设

* 代码规范

  airbnb 代码规范 + eslint + 发布流程规范化

  后期推行typescript

* 未来技术方向

  - PWA + 小程序 + Flutter

  - 小中台 + 微服务

* 技术和平台沉淀

  - 抽离公用组件，形成业务组件库

  - 快速开发快速迭代

* 团队文化

  - 抓大放小，追求极致细节

  - 技术和业务产品融合

  - 不做重复的事情

  - 路走对了就不怕远

* 团队成长

  - 典型代码评审review

  - 月度技术干货分享

  - 团队技术书籍共享

  - 每个成员都有专注的某一细分技术领域和业务领域

* 如何带队

  - 目标

  - 规划

  - 计划

  - 执行

  - 总结
